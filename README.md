# glorkglunk-wavetables

a collection of wavetables to be used on strudel from [Glork Tables 1](https://glorkglunk.gumroad.com/l/pxheC?layout=profile)

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/glorkglunk-wavetables')
```

please [support glorkglunk](https://glitchdotcool.gumroad.com/) if u can
